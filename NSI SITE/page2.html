Les commandes Linux

Ici vous sont présentés 16 différentes commandes Linux.

ls:
affiche par défaut le contenu du répertoire courant.
→Commande ls


mkdir:
permet de créer des repertoires
→Commande mkdir


pwd:
affiche le nom de chemin du répertoire de travail.
→Commande pwd


cd:
passage du répertoire actuel à un autre répertoire
→Commande cd


rmdir:
supprimer du système le répertoire spécifié par le paramètre Directory.
→Commande rmdir


cp:
créer une copie du sommaire du fichier ou du répertoire entier
→Commande cp


rm:
supprime les liens des fichiers ou des répertoires.
→Commande rm


mv:
Déplace les fichiers.
→Commande mkdir


cat:
permet de lire chaque paramètre Fichier dans l'ordre et de l'écrire dans la sortie standard.
→Commande cat


man:
affiche des informations sur les commandes, les sous-routines et les fichiers.
→Commande man


clear:
upprime les actions des commandes z/OS® Debugger précédemment entrées.
→Commande clear


echo:
Inscrit les chaînes de caractères dans la sortie standard.
→Commande echo


ping:
Envoyer une demande d'écho sur un hôte réseau.
→Commande ping


touch:
Met à jour les heures d'accès et de modification d'un fichier.
→Commande touch


sudo:
permet à un autre utilisateur d'exécuter des commandes sur l'hôte cible.
→Commande sudo


chmod:
Permet de modifier les modes d'ouverture de fichier.
→Commande chmod
